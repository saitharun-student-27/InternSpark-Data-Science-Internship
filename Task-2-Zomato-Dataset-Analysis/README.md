# Task 2: Zomato Dataset Analysis

## Objective
The objective of this task is to analyze Zomato restaurant data to identify insights related to restaurant ratings, cuisines, location preferences, restaurant types, online ordering, table booking, price patterns, and factors affecting ratings.

## Dataset
The dataset contains Zomato restaurant information including restaurant name, address, location, ratings, votes, cuisines, restaurant type, online order availability, table booking availability, approximate cost for two people, and listing category.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- WordCloud
- Google Colab
- GitHub

## Work Performed
- Loaded and explored the Zomato dataset
- Cleaned column names
- Cleaned rating and cost columns
- Handled missing values
- Removed duplicate and corrupted rows
- Analyzed rating distribution
- Identified top restaurant locations
- Analyzed online order and table booking impact on ratings
- Explored price vs rating relationship
- Identified popular cuisines and high-rated cuisines
- Created correlation heatmap and cuisine wordcloud
- Prepared recommendations for an Alfido Tech style restaurant platform

## Key Insights
- Average restaurant rating: 3.72
- Most common restaurant location: BTM
- Most popular cuisine: North Indian
- Highest-rated cuisine among common cuisines: Malaysian
- Highest-rated active location: Lavelle Road
- Restaurants with online ordering showed higher average ratings
- Restaurants with table booking showed higher average ratings

## Recommendations
- Focus partnerships in high-density restaurant locations like BTM
- Promote popular cuisines such as North Indian through curated content
- Highlight high-rated niche cuisines such as Malaysian
- Use location-based restaurant recommendations
- Encourage restaurants to support online ordering and table booking
