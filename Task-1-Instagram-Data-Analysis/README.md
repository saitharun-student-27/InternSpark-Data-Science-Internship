# Task 1: Instagram Data Analysis

## Objective
The objective of this task is to analyze Instagram post and engagement data to identify posting patterns, high-performing content types, hashtag performance, and engagement growth opportunities for Alfido Tech.

## Dataset
The dataset contains Instagram-related tables including users, photos, likes, comments, follows, hashtags, and photo-tag mappings.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab
- GitHub

## Analysis Performed
- Loaded and cleaned multiple CSV files
- Standardized column names
- Parsed posting date and time
- Calculated likes, comments, total engagement, follower count, and engagement rate
- Analyzed posting hour and posting day performance
- Compared engagement by content type and filter usage
- Analyzed hashtag usage and hashtag engagement
- Prepared content strategy recommendations for Alfido Tech

## Key Insights
- Posts around 8:00 AM showed the highest average engagement in the available dataset.
- Thursday showed the highest average engagement among the available posting days.
- Photo-based posts performed slightly better than other content types.
- Posts without Instagram filters showed slightly higher engagement.
- The hashtag #delicious showed the highest average engagement.

## Recommendation
Alfido Tech should focus on consistent photo-based posts, especially around the best-performing time window, use relevant hashtags, encourage comments through calls to action, and track engagement rate regularly.

## Limitation
The dataset appears to have limited variation in posting dates and times, so the recommended posting time should be treated as an initial suggestion and tested further with future Instagram data.
