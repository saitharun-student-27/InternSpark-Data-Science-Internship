# Task 1: Instagram Data Analysis

## Objective
The objective of this task is to analyze Instagram post and engagement data to identify the best posting time, high-performing content types, hashtag performance, and engagement improvement strategies for Alfido Tech.

## Dataset
The dataset contains Instagram-related data split across multiple CSV files:
- users.csv
- photos.csv
- likes.csv
- comments.csv
- follows.csv
- tags.csv
- photo_tags.csv

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab
- GitHub

## Analysis Performed
- Loaded and cleaned multiple CSV files
- Standardized column names
- Parsed posting date and time
- Calculated likes, comments, follower count, total engagement, and engagement rate
- Analyzed engagement by posting hour and posting day
- Compared engagement by content type
- Analyzed filter usage impact
- Identified top hashtags by usage and average engagement
- Prepared a recommended content calendar for Alfido Tech

## Key Insights
- Posts at 8:00 AM showed the highest average engagement in the available dataset.
- Thursday showed the highest average engagement among the available posting days.
- Photo-based content performed slightly better than other content types.
- Posts without Instagram filters showed slightly higher engagement.
- The hashtag #delicious showed the highest average engagement.

## Recommendation
Alfido Tech should focus on clean photo-based content, post consistently around the best-performing time window, use relevant hashtags, encourage comments through calls to action, and track engagement rate regularly.

## Limitation
The dataset appears to have limited variation in posting dates and times. Therefore, the recommended posting time should be treated as an initial data-driven suggestion and tested further with future Instagram data.
